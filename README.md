## このソフトウェアについて
自動で、プレゼンテーションのスライドをスクリーンショットし、
指定したディレクトリに保存してくれるソフトウェアです。

スライドを後から見返したい人向けのツールです。

---

## 起動の仕方

`run.py` を実行するだけで起動します。

```bash
python run.py
```

## 設定
 - config.example.json をコピー
 - config.json にリネーム  
 - そして各値を編集

```bash
config.json

{
    "target_display_index": "スクショするディスプレイのインデックス",

    "output_dir_path": "スクリーンショットの保存先ディレクトリ"

    "pixel_diff_value_threshold": "検知する変化の大きさ",

    "tps": "更新速度",
}
```

## 注意

### ■ 他のウィンドウで画面を隠さない

スクリーンショットの対象となる画面は、常に表示された状態にしてください。

### ■ Keyは変更しない

config.json の Key は変更しないことを推奨します。  
変更すると、ソフトウェア内で正常に参照されなくなる可能性があります。
